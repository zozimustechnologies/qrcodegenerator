No test accounts or login required — the extension is fully functional without any sign-in.

To use the extension:
1. Click the extension icon in the toolbar to open the side panel.
2. Type any text or URL into the input field and press Enter or click the QR button.
3. A QR code is generated and displayed instantly. Use the Copy or Download buttons to save it.

Permissions used:
- sidePanel: required to display the UI in the Edge side panel.
- clipboardWrite: required to copy the generated QR code image to the clipboard.

The extension makes a single outbound request to https://api.qrserver.com to generate the QR code image. No user data is stored or transmitted beyond this request. No background scripts run persistently.

No dependencies on other products or services beyond the QR API.
